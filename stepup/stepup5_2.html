教學2 使用標籤

1. 添加輕量標籤

使用tag命令來添加標籤，在 <tagname> 給予標籤的名稱。

$ git tag <tagname>

在 HEAD 指向的提交裡增加名為 "apple" 的標籤，請執行以下的命令。 

$ git tag apple

如果僅使用tag命令，不添加其他參數，則會顯示標籤列表。

$ git tag
apple

在 log 命令添加 --decorate 執行，可以顯示包含標籤資料的歷史記錄。 

$ git log --decorate
commit e7978c94d2104e3e0e6e4a5b4a8467b1d2a2ba19 (HEAD, tag: apple, master)
Author: yourname <yourname@yourmail.com>
Date:   Wed Jul 18 16:43:27 2012 +0900

    first commit
