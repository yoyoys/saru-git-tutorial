在開發分支上修改

Backlog篇

1. 建立分支

首先，來建立作業用的分支吧。因為是排序陣列的處理，所以建立一個名為「add-sort-func」的分支。 

$ git checkout -b add-sort-func
「建立分支」請參考這裡

2. 修改檔案並提交

已經切換到作業分支了，請將檔案修改如下。

sort.jsvar sortNumber = function (number) {
   number.sort(function (a, b) {
       if (a == b) {
           return 0;
       }
       return a < b ? -1 : 1;
   });
};

var number = [19, 3, 81, 1, 24, 21];
sortNumber(number);
console.log(number);

修改完成後進行提交。

$ git add sort.js
$ git commit -m "新增了排序陣列的處理"

3. Push修改後的分支

提交完成後Push到遠端分支。

$ git push origin add-sort-func
