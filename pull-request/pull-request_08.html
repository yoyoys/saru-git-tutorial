教學 來使用Pull Request吧

從畫面上進行合併

Backlog篇

審核後如果沒有問題就進行合併。請點擊「合併」按鈕。

會顯示以下畫面。請給開發者附上適當的評論後進行合併。另外，對開發者來說被合併是一件很開心的事。請務必附上感謝的話語來留下評論。

合併後會顯示以下畫面。狀態會從 [ Open ] 變為 [ Merged ]，並且會顯示「刪除分支」按鈕。不再需要的開發分支可以從這裡刪除。

查看「網路」分頁，可以確認開發分支已經合併到主分支。

GitHub篇

審核後如果沒有問題就進行合併。請點擊「Conversation」分頁的「Merge pull request」按鈕。

會顯示以下畫面。請附上適當的評論後，點擊「Confirm merge」進行合併。另外，對開發者來說被合併是一件很開心的事。請務必附上感謝的話語來留下評論。

合併後會顯示以下畫面。會出現「Delete branch」按鈕。不再需要的開發分支可以從這裡刪除。

查看「Graphs」選單的「Network」分頁，可以確認開發分支已經合併到主分支。
