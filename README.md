# 連猴子都能懂的 Git 入門指南

> **線上閱讀：<https://yoyoys.github.io/saru-git-tutorial/>**

這是來自網路上相當有名的「連猴子都能懂的 Git 入門指南」，由 [Backlog（貝格樂）](https://backlog.com/) 所製作的 Git 教學網站。由於原本網站已經消失，所以將其整理完善後放在 GitHub 上面，此外也加入了後來日文版才有的 Pull Request 環節。

## 內容

- **入門篇** — Git 的基本概念與操作
- **進階篇** — 分支、合併等進階操作
- **Pull Request** — 日文版新增的 Pull Request 教學
- **Git 命令快速參考** — 常用 Git 命令速查表
